About

Application Buddy is a free web app built using replit, Boostrap, HTML, CSS & JS. The download includes the complete source files including HTML, CSS, and JavaScript as well as optional SASS stylesheets for easy customization.

As a job seeker myself, I know how frustrating filling out job apps can be. So hopefully this web app can help make things just a little bit easier. I drink home brew coffee, but I'll leave this here anyway.

You can create your own Application Buddy by simply typing in your information and pressing enter to commit! This will automatically send you to the next input for a seamless flow.

There is a 'add button' in case you need another input field that comes with fields for a label and information. Jobs and References have an extra button that will copy the whole card in case you need multiple jobs or references.
